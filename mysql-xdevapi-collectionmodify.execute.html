CollectionModify::execute — 変更の操作を実行する

説明

public mysql_xdevapi\CollectionModify::execute ( void ) : mysql_xdevapi\Result

MySQLサーバに CRUD 操作のリクエストを送信するために、 execute メソッドは必須です。

パラメータ

この関数にはパラメータはありません。

返り値

影響を受けた行数のような、操作の状態を確認するために使われる Result オブジェクトを返します。
