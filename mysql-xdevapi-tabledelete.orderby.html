TableDelete::orderby — Set delete sort criteria

説明

public mysql_xdevapi\TableDelete mysql_xdevapi\TableDelete::orderby ( string $orderby_expr )

Set the order options for a result set.

パラメータ

orderby_expr

The sort definition.

返り値

A TableDelete object.

例

例1 mysql_xdevapi\TableDelete::orderBy() example

<?php
$session = mysql_xdevapi\getSession("mysqlx://user:password@localhost");

$schema = $session->getSchema("addressbook");
$table  = $schema->getTable("names");

$table->delete()
  ->where("age = :age")
  ->bind(['age' => 42])
  ->orderby("name DESC")
  ->limit(1)
  ->execute();

?>
