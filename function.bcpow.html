bcpow

(PHP 4, PHP 5, PHP 7)

bcpow — 任意精度数値をべき乗する

説明

bcpow ( string $base , string $exponent [, int $scale = 0 ] ) : string

base の exponent 乗を求めます。

パラメータ

base

基数を表す文字列。

exponent

指数を表す文字列。 指数が整数でない場合、値は切り捨てられます。 正しい指数の範囲はプラットフォーム依存ですが、 少なくとも -2147483648 から 2147483647 までの範囲を持ちます。

scale

このオプションパラメータを使用して、 結果の小数点以下の桁数を指定します。省略した場合は、bcscale() 関数でグローバルに設定した桁数をデフォルトとして使用します。それも設定されていない場合は 0 を使用します。

返り値

結果を文字列で返します。

変更履歴

バージョン 説明
7.3.0 bcpow() now returns numbers with the requested scale. Formerly, the returned numbers may have omitted trailing decimal zeroes.

例

例1 bcpow() の例

<?php

echo bcpow('4.2', '3', 2); // 74.08

?>

注意

注意:

PHP 7.3.0 より前のバージョンでは、bcpow() の返す結果の小数点以下の桁数は、 scale パラメータで指定したものより少なくなる可能性がありました。 結果を表すのに scale で指定した桁数が不要な場合にのみそうなります。たとえば次のような場合です。

例2 bcpow() での桁数の例

<?php
echo bcpow('5', '2', 2);     // 結果は "25.00" ではなく "25" となります
?>
