はじめに

この拡張モジュールは、MySQLドキュメントストアに X DevAPI 経由でアクセスできるようにします。 X DevAPI は、複数の MySQLコネクタ によって提供されている、共通のAPIです。 これらのコネクタは、リレーショナルなテーブルだけでなく、JSONで表現されたドキュメントの集合に対しても、 CRUDスタイルの操作を持つAPIを通して簡単にアクセスできます。

X DevAPI は Xプロトコルを使っています。 これは MySQL 8.0 サーバの、新世代のクライアント-サーバプロトコルです。

MySQLドキュメントストアに関する一般的な情報については、 MySQLマニュアル の » MySQL Document Store の章を参照してください。
