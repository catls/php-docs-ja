Parle\Parser::reduceId

(PECL parle >= 0.5.1)

Parle\Parser::reduceId — Get reduce id

説明

public int Parle\Parser::reduceId ( void )

Get the grammar rule id just processed in the reduce action. The returned value corresponds either to a token or to a parser rule.

パラメータ

この関数にはパラメータはありません。

返り値

Returns integer represeting the rule id.
