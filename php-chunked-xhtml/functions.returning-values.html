返り値

オプションの return 文により値を返すことができます。 配列やオブジェクトを含むあらゆる型を返すことができます。 これにより、関数の実行を任意の箇所で終了し、その関数を呼び出した 箇所に制御を戻すことが出来ます。詳細に関しては return を参照してください。

バージョン 説明
7.0.0 戻り値の型宣言が追加されました。

注意:

return を省略した場合は NULL を返します。

return の使いかた

例1 return の使用法

<?php
function square($num)
{
    return $num * $num;
}
echo square(4);   //  '16'を出力
?>

複数の値を返すことはできませんが、リストを返すことにより 同じ効果を得ることができます。

例2 複数の値を得るために配列を返す

<?php
function small_numbers()
{
    return array (0, 1, 2);
}
list ($zero, $one, $two) = small_numbers();
?>

関数からリファレンスを返すには、リファレンス演算子 & を関数宣 言部および変数への返り値を代入する際の両方で使用する必要があります。

例3 関数からリファレンスを返す

<?php
function &returns_reference()
{
    return $someref;
}

$newref =& returns_reference();
?>

リファレンスに関するさらに詳しい情報がリファレンスの説明にあります。

戻り値の型宣言

PHP 7 以降で、戻り値の型宣言が導入されました。関数の戻り値が定義済みの型になるよう強制できるようになります。 これを有効にするには、 declare(strict_types=1) と設定しなければいけません。 このディレクティブが 0 であったり未設定であったりする場合は、関数に対して弱い型チェックをするようになります。 このディレクティブが有効なのはそれを記入したファイル内で宣言した関数だけであり、 このファイルからインクルードした別のファイルには影響しません。 また、他のファイルからこのファイルをインクルードした場合にも、インクルード元のファイルには影響しません。 関数が呼び出し元がどこであるかは、気にする必要はありません。 宣言できる型は Classe、Interface、Array、Int、Float、String、Bool です。

例4 strict_types ディレクティブの影響

<?php
declare(strict_types=1); // file1.php

// file 1 - 厳密な型チェック
// file 2 - 弱い型チェック

include "file2.php";

var_dump(a()); // 正しく動作します
var_dump(b()); // TypeError となります


function b() : int{
   return "1";  
}
?>


<?php
// file2.php
function a(): int{
   return "1";
}

?>

例5 戻り値の型

関数のシグネチャは、親クラスでの宣言と一致する必要があります。親が戻り値の型を宣言していない場合は、 子の側で型を宣言できます。

<?php
declare(strict_types=1);

interface Numbers{
  abstract public function one() : int ;
  abstract public function two();         // 戻り値の型を定義していません
}

class NumClass implements Numbers
{
  public function one()  : int {  // 戻り値の型は抽象クラスと同じ
    return 0;
  }

  public function two()  {  // 親の抽象クラスの関数で、
    return 2;               // 戻り値の型が定義されていません
  }
}

class display extends NumClass
{
  public function one() : int // 戻り値の型は親クラスと同じ
  {
    return 1;
  }

  public function two() : int // 戻り値の型は親クラスと同じ
  {
    return 2;
  }
}

$dis = new display();
var_dump($dis->one());
var_dump($dis->two());

?>

例6 クラスを戻す

<?php
declare(strict_types=1);

interface A {
    static function make();
}
class B implements A {
    static function make(): B { // 有効
        return new B();
    }
}

例7 TypeError

戻り値の型が不正な場合は TypeError となります。

<?php
declare(strict_types=1);

function show(): int{
    return "1";
}

try {
    show();    
} catch (TypeError $e) {
 var_dump($e->getMessage()); // string(121) "Return value of show() must be of the type integer, 
                             // string returned in %s on line 5"
}
?>
