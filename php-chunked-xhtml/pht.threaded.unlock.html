Threaded::unlock

(PECL pthreads < 3.0.0)

Threaded::unlock — Unlocks a mutex lock

説明

public void Threaded::unlock ( void )

This method will unlock the mutex lock associated with the given class (either a HashTable, Queue, Vector, or AtomicInteger).

パラメータ

この関数にはパラメータはありません。

返り値

No return value.
