Parle\Lexer::bol

(PECL parle >= 0.5.1)

Parle\Lexer::bol — Retrieve or set the flag indicating start of input

説明

public mixed Parle\Lexer::bol ([ boolean $bol ] )

This methods sets or retrieves the flag value indicating the start of input or the the last character read being \n.

パラメータ

bol

Flag value to be set.

If a new value is passed, NULL is returned. Otherwise boolean is returned.
