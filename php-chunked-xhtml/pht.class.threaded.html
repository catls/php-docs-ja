The Threaded interface

(PECL pthreads >= 2.0.0)

はじめに

The Threaded interface is an internal interface used by the Inter-Thread Communication (ITC) data structures (HashTable, Queue, and Vector). It allows those data structures to be threaded and ensures that the mutex locking API (Threaded::lock() and Threaded::unlock()) is implemented by each of the ITC data structures. It is not implementable by userland classes (since standalone mutex locks are not exposed).

インターフェイス概要

Threaded {
/* メソッド */
public void lock ( void )
public void unlock ( void )
}
