Changes to INI File Handling

Removed Directives

  • Removed asp_tags ini directive. Trying to enable it will result in a fatal error.
  • Removed always_populate_raw_post_data ini directive

Removed #-Style Comments

The deprecated # comments have now been removed. Comments can now only be specified using the semi-colon.
