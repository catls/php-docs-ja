is_countable

(PHP 7 >= 7.3.0)

is_countable — Verify that the contents of a variable is a countable value

説明

bool is_countable ( mixed $var )

Verify that the contents of a variable is an array or an object implementing Countable

パラメータ

var

The value to check

返り値

Returns TRUE if var is countable, FALSE otherwise.

変更履歴

バージョン 説明
7.3.0 is_countable() has been added.

例

例1 is_countable() examples

<?php
var_dump(is_countable([1, 2, 3])); // bool(true)
var_dump(is_countable(new ArrayIterator(['foo', 'bar', 'baz']))); // bool(true)
var_dump(is_countable(new ArrayIterator())); // bool(true)
var_dump(is_countable(new stdClass())); // bool(false)

参考

  • is_object() - 変数がオブジェクトかどうかを検査する
  • is_iterable() - 変数の内容が反復可能な値であることを確認する
  • is_bool() - 変数が boolean であるかを調べる
