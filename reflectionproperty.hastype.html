ReflectionProperty::hasType

(PHP 7 >= 7.4.0)

ReflectionProperty::hasType — プロパティが型を持つかをチェックする

説明

public ReflectionProperty::hasType ( void ) : bool

プロパティが、自身に関連付けられた型を持つかをチェックします。

パラメータ

この関数にはパラメータはありません。

返り値

型が指定されていれば TRUE そうでなければ FALSE

例

例1 ReflectionProperty::hasType() の例

<?php
class User
{
    public string $name;
}

$rp = new ReflectionProperty('User', 'name');
var_dump($rp->hasType());
?>

上の例の出力は以下となります。

bool(true)
